# PRD：拜访单自动生成

## 1. 背景与目标
通过 AI 将拜访录音、图片、邮件等材料转化为拜访单草稿，降低一线人员填写成本，提高拜访记录质量。

## 2. 范围
本期支持 AI 生成草稿、人工确认、字段校验、提交。

## 3. 非范围
本期不支持自动提交、不替代人工确认、不自动修改客户主数据。

## 4. 核心功能
| 功能 | 描述 | 优先级 |
|---|---|---|
| 材料上传 | 支持上传录音、图片、邮件文本 | P0 |
| AI 草稿生成 | 生成拜访单字段草稿 | P0 |
| 人工确认 | 支持编辑、确认、提交 | P0 |
| 置信度提示 | 展示低置信度字段 | P1 |
| 修改回流 | 记录人工修改结果 | P1 |

## 5. AI 能力设计
AI 只做辅助生成，不直接提交。所有输出需要人工确认。
